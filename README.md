#How to use:
* First, include the jquery.acessibility.js script inside your page, then:
* You just call the plugin, like this: $("#container").fontSize(); and it works with the default settings!
* If you don,t want to use the custom value, you can change it, just like this: $("#container").fontSize({"minSize" : "5", "maxSize" : "22"});

#Default options:
* size : 12
* target : "p"
* maxSize : 17
* minSize : 9
* speed : 100
